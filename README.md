# Lichtenberg Algorithms ⚡

Physics-inspired metaheuristics based on lightning propagation patterns.

## 🚀 Includes

- **Lichtenberg Algorithm (LA)** – single-objective optimization
- **Multi-objective Lichtenberg Algorithm (MOLA)** – Pareto-based optimization

## 📖 References

- Pereira, J. L. J., Francisco, M. B., Diniz, C. A., Oliver, G. A., Cunha Jr, S. S., & Gomes, G. F. (2021). Lichtenberg algorithm: A novel hybrid physics-based meta-heuristic for global optimization. Expert Systems with Applications, 170, 114522.
- Pereira, J. L. J., Oliver, G. A., Francisco, M. B., Cunha Jr, S. S., & Gomes, G. F. (2022). Multi-objective lichtenberg algorithm: A hybrid physics-based meta-heuristic for solving engineering problems. Expert Systems with Applications, 187, 115939.

## ⚡ Motivation

Inspired by lightning propagation and Lichtenberg figures, the algorithm explores the search space using fractal patterns, dynamically balancing exploration and exploitation.
